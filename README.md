# Customer Support Ticket System

A web-based customer support ticket management system built with FastAPI, React, Tailwind CSS, and SQLite.

## Features

- Create support tickets
- View all tickets
- Search tickets
- Filter by status
- Update ticket status
- Add notes/comments
- Delete tickets

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- FastAPI
- SQLAlchemy

Database:
- SQLite

Deployment:
- Render

## API Endpoints

POST /api/tickets

GET /api/tickets

GET /api/tickets/{ticket_id}

PUT /api/tickets/{ticket_id}

DELETE /api/tickets/{ticket_id}
